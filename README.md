# Project Athena (Working Title for now)

An AI-powered Study & Wellness Agent built using Java, Spring Boot, MySQL, and LLM APIs.

## Vision

Build an intelligent assistant that helps students:
- Plan their studies
- Track progress
- Adapt schedules based on workload
- Monitor well-being through daily check-ins
- Provide personalized AI guidance

## Tech Stack

- Java 21
- Spring Boot
- MySQL
- REST API
- Git & GitHub
- OpenAI API (or compatible LLM)
